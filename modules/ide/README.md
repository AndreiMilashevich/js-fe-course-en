## Module Description

Students are required to get acquainted with:

- The functioning and nuances of browsers
- Features of Chrome Dev Tools (Styles and Element Inspector)
- The VSCode Editor
    - Setting up the environment
    - Extensions
    - Console
- Basic terminal commands
