# CSS Modules And Some CSS New Features

## In this module, students need to:

Know and understand:

- CSS Modules Approach
- Container Queries
- Style Queries
- CSS Nesting
- New Viewport Units
- Selector :has()
- nth-of Syntax
- Popover
- Anchor Positioning
- CSS color-mix()
- Trigonometric Functions In CSS
